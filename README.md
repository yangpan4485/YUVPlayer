# YUVPlayer
Windows YUV 播放器

使用 duilib 开发的 YUV 播放器，操作简单、实用

## 编译命令

+ md build
+ cd build
+ cmake ..
+ 使用 VS 打开 player.sln 运行即可

## 下载安装包程序

[安装包下载](https://raw.githubusercontent.com/yangpan4485/YUVPlayer/main/YUVPlayerInstall.exe)
